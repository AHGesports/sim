# Agent Data System - UI/UX Specification v1.0

## Overview

This document defines the UI/UX specification for the Agent Data System, where **Workspaces = Agents**. Each agent has isolated data, workflows, and MCP configurations, with a shared Global scope for cross-agent resources.

---

## Core Concepts

| Concept | Definition |
|---------|------------|
| **Agent** | A workspace with isolated data, workflows, and MCP servers |
| **Global** | Shared scope accessible by all agents (read/write) |
| **Data Explorer** | Interface for managing database tables and schemas |
| **MCP** | Model Context Protocol servers for agent tools/functions |
